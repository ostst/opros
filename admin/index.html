Графики итогов — только на странице для большого экрана (ссылка ниже). Кнопка «Показать результат на большом экране» открывает итоги там. Кнопка «Случайный розыгрыш: 3 номера» сохраняет три числа и показывает их на том же экране (крупно); на телефонах не дублируется. Число сверху — завершившие опрос; строка ниже — уникальные заходы по ссылке.

Открыть экран «Результаты опроса» (для проектора)

Приветственный экран

Текст до кнопки «Начать опрос» на странице опроса. В заголовке новая строка = перенос. В первом абзаце можно написать {{nq}} — подставится число вопросов (жирным). Участники по умолчанию могут отмечать несколько вариантов на вопрос — учтите это в формулировке приветствия.

Вопросы опроса

У каждого вопроса — текст и любое число вариантов (не меньше двух). По умолчанию участник может отметить несколько вариантов; снимите галочку «Несколько вариантов», если нужен только один ответ. После изменений нажмите «Сохранить вопросы». Счётчики показывают, сколько раз выбрали каждый вариант (при множественном выборе сумма по вопросу может быть больше числа анкет).
